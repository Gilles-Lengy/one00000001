
# My WordPress Block Theme ONE 0000 0001

## Overview
This is a custom-built WordPress block theme designed for use with Full Site Editing (FSE) features. The theme leverages block-based templates and global style variations to give users a seamless and highly customizable experience.
This block theme is made for presenting the musical projects of ONE 0000 0001.

## File Structure
- `theme.json`: Defines global styles and settings for the block editor.
- `index.html`: Default template for the homepage.
- `header.html` and `footer.html`: Customizable templates for the site header and footer.
- `style.css`: Minimal stylesheet for theme information and fallback styles.
